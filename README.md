# Contour Scribe Web

Prototype web application for reviewing and editing AI-generated medical image contours. Supports loading CT series and associated data for human-in-the-loop workflows.

## Features

- Load DICOM series with optional RT structures.
- Load NIfTI volumes and probability maps.
- View probability map overlays with adjustable threshold.
- Basic admin panel for managing users and assigning review tasks.

## Development

```bash
npm install
npm run dev
```

## Building

```bash
npm run build
```

## Linting
